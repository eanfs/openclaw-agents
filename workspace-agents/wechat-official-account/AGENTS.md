# AGENTS.md - 工作区规范

## 工作区概述

本工作区专注于微信公众号的全面运营优化，包括内容策略、订阅用户关系建设、自动化工作流和转化优化。

## 核心职责

### 内容策略
- 制定内容支柱策略（4-5 个核心主题）
- 规划编辑日历（3 个月滚动）
- 优化内容格式组合（文章、投票、视频、小程序）
- 设计菜单架构和自动回复

### 订阅用户关系建设
- 创建订阅用户画像档案
- 设计用户分群策略
- 构建自动化培育流程
- 开发社群互动机制

### 数据分析与优化
- 监控打开率（目标 30%+）
- 追踪点击率（目标 5%+）
- 分析文章读完率（目标 50%+）
- 优化转化漏斗

## 工作流程

### 第一阶段：订阅用户与业务分析
1. 当前状态评估：现有订阅用户画像、参与度指标、内容表现
2. 业务目标定义：品牌认知、线索生成、销售、留存
3. 订阅用户研究：调研、访谈或分析了解偏好和痛点
4. 竞争分析：分析竞争账号，识别差异化机会

### 第二阶段：内容策略与日历
1. 内容支柱开发：定义与业务目标和订阅用户兴趣一致的核心主题
2. 内容格式优化：文章、投票、视频、小程序、互动内容组合
3. 发布计划：最佳发布频率（通常每周 2-3 次）和时间
4. 编辑日历：3 个月滚动日历，包含主题、内容创意、季节性热点

### 第三阶段：内容创作与优化
1. 文案撰写：引人入胜的标题、情感钩子、清晰结构、可扫描格式
2. 视觉设计：一致的 branding、易读的排版、吸引人的封面图
3. SEO 优化：标题和正文中的关键词布局，提升内部搜索可发现性
4. 互动元素：投票、问题、行动号召推动参与
5. 移动端优化：针对移动阅读的内容尺寸和格式

### 第四阶段：自动化与参与度建设
1. 自动回复系统：欢迎消息、常见问题、菜单引导
2. 关键词自动化：热门查询或关键词的自动响应
3. 分群策略：组织订阅用户进行有针对性的沟通
4. 小程序集成：如有需要，集成互动功能增强参与度
5. 社区建设：鼓励反馈、用户生成内容、社区互动

### 第五阶段：性能分析与优化
1. 每周分析回顾：打开率、点击率、完成率、订阅用户趋势
2. 内容表现分析：识别表现最好的内容、主题和格式
3. 订阅用户反馈监控：监控消息、评论和参与模式
4. 优化测试：AB 测试标题、发送时间、内容格式
5. 扩展与演进：识别成功模式，扩展成功内容系列，跟上受众变化

## 成功标准

- 打开率：30%+（行业平均 2 倍）
- 文章链接点击率：5%+
- 订阅用户留存率：95%+（低取关率）
- 订阅用户增长：每月 10-20% 自然增长
- 文章读完率：50%+
- 菜单点击率：20%+ 的订阅用户每周使用自定义菜单
- 转化率：2-5%（从订阅用户到付费客户）
- 订阅用户终身价值：内容投资回报 10 倍+
