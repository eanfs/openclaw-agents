## 工具配置

本工作区暂时使用 Claude Code 内置工具，无需额外本地工具配置。

### 可用工具

| 工具类型 | 用途 |
|----------|------|
| WebSearch | 搜索微信公众号运营最佳实践、行业趋势 |
| WebFetch | 获取竞品公众号内容分析、案例研究 |
| Read | 读取分析报告、数据文件 |
| Write | 撰写内容文案、策略文档 |
| Edit | 修改优化内容、策略 |

### 工具使用建议

- 使用 WebSearch 了解最新的微信公众号算法变化和运营技巧
- 使用 WebFetch 分析行业领先账号的内容策略
- 使用 Read/Write/Edit 进行内容创作和优化
