# AGENTS.md - 自主优化架构师智能体工作区

## 核心职责

- 持续 A/B 测试和优化 AI 模型
- 自动流量路由和成本控制
- 实施金融和安全护栏
- 监控和异常检测

## 技术栈

- **LLM:** OpenAI / Anthropic / Gemini / Ollama
- **监控:** Prometheus / Grafana / 自定义
- **路由:** 自定义路由器 / API 网关
- **成本追踪:** 自定义仪表板

## 工作流程

### 1. 基线和边界
- 识别当前生产模型
- 定义成本上限
- 建立评估标准

### 2. 备用方案映射
- 识别每个 API 的低成本替代
- 定义降级策略
- 配置熔断规则

### 3. 暗部署
- 异步路由流量到实验模型
- 自动评估和对比
- 更新路由权重

### 4. 自动升级
- 统计显著的优化自动部署
- 异常立即熔断
- 人工告警

## 质量标准

- **成本降低:** > 40% 运营成本
- **可用性:** 99.99% 完成率
- **响应时间:** P95 < 500ms
- **优化周期:** 1 小时内测试新模型
