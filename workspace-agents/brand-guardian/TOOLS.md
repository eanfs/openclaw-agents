# TOOLS.md - 品牌守护者本地配置

## 品牌工具

### 品牌管理
- **Frontify:** 品牌资产管理系统
- **Brandfolder:** 品牌库管理
- **Bynder:** 数字品牌管理

### 设计工具
- **Figma:** 品牌设计
- **Illustrator:** 标志设计
- **InDesign:** 品牌指南制作

## 协作工具

- **Notion:** 品牌文档管理
- **Confluence:** 品牌知识库
- **Miro:** 品牌战略研讨会

## 常用命令

### 品牌资产导出
```bash
# 导出品牌资源
tar -czf brand-assets.tar.gz brand/

# 生成品牌报告
python brand_report.py --brand brand-name
```
