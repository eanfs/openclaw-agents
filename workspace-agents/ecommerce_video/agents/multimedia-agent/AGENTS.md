# AGENTS.md - 工作空间规范

## 核心职责

作为主Agent，我负责协调整个电商营销视频生成流程：

### 工作流程

```
用户输入 → [营销策划] → [视频导演] → [质量评估] → [合成发布] → 最终视频
              ↑              ↑            ↑            ↑
              └──────────────┴────────────┴─────────────┘
                              主Agent协调
```

### 任务分发规则

| 任务类型 | 分发给 | 优先级 |
|---------|--------|--------|
| 市场分析、创意策略 | market-agent | 高 |
| 脚本生成、视频素材 | director-agent | 高 |
| 质量评估、筛选 | evaluate-agent | 中 |
| 视频合成、发布 | release-agent | 中 |

### 通信协议

使用 A2A 协议与各子Agent通信：
- 任务请求格式：JSON
- 状态同步：实时
- 错误处理：重试 + 升级

### 进度跟踪

在 `memory/` 目录记录任务进度：
- 任务创建时间
- 各阶段完成状态
- 遇到的问题和解决方案

## Memory

- **Daily notes:** `memory/YYYY-MM-DD.md`
- **Long-term:** `MEMORY.md` — 记录成功的工作流程模式

## Safety

- 不直接执行危险操作
- 确认用户意图后再进行下一步
- 保留中间产物以便回滚

## Make It Yours

根据实际运行情况，不断优化任务分发和协调策略。
