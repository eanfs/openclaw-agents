# TOOLS.md - 工具配置

## 评估工具

### 技术检测
- FFprobe：视频编码、分辨率、帧率检测
- PaddleSpeech：音频质量分析
- TensorFlow：画质评估模型

### 内容分析
- GPT/Vision：内容理解和逻辑分析
- CLIP：图文一致性检测
- 自研模型：吸引力评分

### 合规检测
- 敏感词库：违规词汇检测
- 版权检测：图片指纹比对
- 平台规则：各平台规范检查

## 评估模板

### 技术质量检查表
```markdown
## [素材ID] 技术评估
- [ ] 分辨率 ≥ 1080p
- [ ] 帧率 ≥ 24fps
- [ ] 音频清晰无杂音
- [ ] 音量 -20dB ~ -6dB
- [ ] 无明显卡顿
- [ ] 编码格式正确
```

### 内容质量检查表
```markdown
## [素材ID] 内容评估
- [ ] 黄金3秒开头
- [ ] 核心卖点突出
- [ ] 产品展示清晰
- [ ] 行动号召明确
- [ ] 时长合适
- [ ] 节奏恰当
- [ ] 无逻辑错误
```

### 合规性检查表
```markdown
## [素材ID] 合规检查
- [ ] 无敏感词
- [ ] 无违规内容
- [ ] 版权合规
- [ ] 平台规则符合
- [ ] 广告法合规
- [ ] 品牌展示规范
```

## 评分权重

### 抖音平台
- 技术质量：25%
- 内容质量：45% (吸引力权重高)
- 合规性：30%

### 小红书平台
- 技术质量：30% (画质要求高)
- 内容质量：40%
- 合规性：30%

### 快手平台
- 技术质量：25%
- 内容质量：45% (转化权重高)
- 合规性：30%
