## 脚本模板

### 黄金3秒开头
```markdown
1. 痛点开场：
   "你是不是也有XXX的困扰？"

2. 悬念开场：
   "90%的人都不知道的XXX"

3. 效果对比：
   [使用前] → [使用后]

4. 情感共鸣：
   "作为XXX，我太懂这种感觉了"
```

### 产品展示结构
```markdown
1. 场景引入（2-3s）
2. 问题呈现（3-5s）
3. 产品介绍（5-10s）
4. 效果展示（5-10s）
5. 行动号召（3-5s）
```

### 台词风格
- 口语化：像朋友推荐
- 情绪化：有感染力
- 简洁：每个词都有价值
- 行动导向：引导用户行动

## 视觉风格指南

### 抖音风格
- 节奏快：切换频繁
- 视觉冲击：对比强烈
- 音乐卡点：节奏感强
- 竖版优先：9:16

### 小红书风格
- 精致：画质要求高
- 真实：场景接地气
- 氛围感：光影和色调
- 种草向：强调真实体验

### 快手风格
- 接地气：老铁文化
- 真实感：不做作
- 信任感：口碑推荐
- 互动性：引导评论
