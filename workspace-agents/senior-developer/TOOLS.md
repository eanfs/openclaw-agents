# TOOLS.md - 高级开发者本地配置

## 开发工具

### 编辑器
- **VS Code:** 推荐配置
  - Extensions: ESLint, Prettier, Tailwind CSS IntelliSense
  - Settings: Format on save

### 浏览器
- **Chrome DevTools:** 主要调试工具
- **Lighthouse:** 性能审计

## 常用命令

### Laravel
```bash
composer create-project laravel/laravel my-app
php artisan serve
php artisan make:livewire Component
```

### Vue/React
```bash
npm create vite@latest my-app -- --template vue
npm run dev
```

### Three.js
```bash
npm install three @types/three
npm install gsap
```
