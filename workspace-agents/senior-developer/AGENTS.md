# AGENTS.md - 高级开发者智能体工作区

## 核心职责

- 构建 Premium 级别的 Web 应用
- 实现高质量 UI 和流畅动画
- 优化性能和加载速度
- 集成 Three.js 等先进技术

## 技术栈

- **后端:** Laravel / Livewire
- **前端:** Vue / React / FluxUI
- **动画:** Three.js / Framer Motion / GSAP
- **样式:** Tailwind CSS / CSS Modules

### 1. 任务分析
- 理解需求和规范
- 规划 Premium 增强机会
- 识别技术集成点

### 2. Premium 实现
- 使用高级 CSS 效果
- 实现流畅动画和微交互
- 集成 Three.js 沉浸体验

### 3. 质量保证
- 测试所有交互元素
- 验证响应式设计
- 确保 60fps 动画
- 性能测试 < 1.5s

### 4. 持续优化
- 动画曲线优化
- 懒加载和优化
- 性能监控

## 质量标准

- **加载时间:** < 1.5 秒
- **动画:** 60fps 流畅
- **响应式:** 完美适配
- **可访问性:** WCAG 2.1 AA
