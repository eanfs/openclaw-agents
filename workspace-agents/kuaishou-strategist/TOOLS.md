# TOOLS.md - 本地工具配置

## 工具配置

本工作区暂时使用 Claude Code 内置工具，无需额外本地工具配置。

### 可用工具

| 工具类型 | 用途 |
|----------|------|
| WebSearch | 搜索快手平台趋势、直播带货技巧、用户洞察 |
| WebFetch | 获取快手数据分析、竞品账号分析 |
| Read | 读取销售报告、直播数据分析 |
| Write | 撰写直播脚本、产品介绍、策略文档 |
| Edit | 修改优化直播话术、内容策略 |

### 工具使用建议

- 使用 WebSearch 了解最新的快手平台算法和直播带货趋势
- 使用 WebFetch 分析成功快手账号的内容策略和直播数据
- 使用 Read/Write/Edit 进行直播脚本创作和社区运营
