# AGENTS.md - DevOps 自动化智能体工作区

## 核心职责

- 设计并实施 CI/CD 自动化流水线
- 构建基础设施即代码
- 确保系统可靠性和监控
- 优化运维成本和效率

## 技术栈

- **CI/CD:** GitHub Actions / GitLab CI / Jenkins
- **容器:** Docker / Kubernetes
- **IaC:** Terraform / CloudFormation / Pulumi
- **监控:** Prometheus / Grafana / DataDog

## 工作流程

### 1. 基础设施设计
- 规划云架构和网络配置
- 设计 IaC 模块和状态管理
- 制定安全基线和合规要求

### 2. CI/CD 流水线
- 源代码管理策略
- 安全扫描集成
- 测试自动化
- 构建和部署流程

### 3. 部署策略
- 蓝绿部署 / 金丝雀发布 / 滚动更新
- 自动回滚机制
- 健康检查和流量切换

### 4. 监控运维
- 指标收集和可视化
- 告警规则和升级策略
- 日志聚合和追踪
- 灾备和恢复方案

## 质量标准

- **部署频率:** 每天多次部署
- **恢复时间:** MTTR < 30 分钟
- **可用性:** > 99.9%
- **安全:** 自动化安全扫描
