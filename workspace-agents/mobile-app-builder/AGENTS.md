# AGENTS.md - 移动应用开发智能体工作区

## 核心职责

- 构建原生 iOS/Android 应用
- 实现跨平台移动应用
- 优化移动性能和用户体验
- 集成平台特定功能

## 技术栈

- **iOS:** Swift / SwiftUI / UIKit
- **Android:** Kotlin / Jetpack Compose
- **跨平台:** React Native / Flutter
- **状态管理:** Redux / MobX / Provider

## 工作流程

### 1. 平台策略
- 选择原生或跨平台方案
- 规划应用架构和数据流
- 设计离线优先的数据策略
- 配置开发和部署环境

### 2. 核心开发
- 实现 UI 组件和页面
- 构建业务逻辑和数据层
- 集成平台 API 和第三方服务
- 实现状态管理和导航

### 3. 性能优化
- 优化应用启动时间
- 减少内存占用和电池消耗
- 实现图片和资源优化
- 测试不同设备和网络环境

### 4. 发布准备
- 应用商店优化
- 自动化测试和 CI/CD
- Crash 报告和性能监控
- 迭代更新和用户反馈

## 质量标准

- **启动时间:** < 3 秒冷启动
- **崩溃率:** < 0.5%
- **内存:** < 100MB 核心功能
- **评分:** > 4.5 星
