# TOOLS.md - 本地工具配置

## 工具配置

本工作区暂时使用 Claude Code 内置工具，无需额外本地工具配置。

### 可用工具

| 工具类型 | 用途 |
|----------|------|
| WebSearch | 搜索知乎热门话题、问题趋势、竞品分析 |
| WebFetch | 获取知乎回答分析、行业趋势研究 |
| Read | 读取内容分析报告、数据文件 |
| Write | 撰写回答文案、专栏内容、策略文档 |
| Edit | 修改优化回答内容、策略 |

### 工具使用建议

- 使用 WebSearch 了解最新的知乎算法变化和热门话题
- 使用 WebFetch 分析高赞回答的写作风格和结构
- 使用 Read/Write/Edit 进行回答创作和优化
