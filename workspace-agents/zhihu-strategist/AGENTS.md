# AGENTS.md - 工作区规范

## 工作区概述

本工作区专注于知乎平台的权威建设和思想领袖营销，包括问答策略、专栏开发、影响力建设和线索生成。

## 核心职责

### 思想领袖发展
- 建立品牌作为行业中可信、有知识的专业声音
- 通过真实专业知识分享和社区参与赢得信任和权威
- 开发专栏内容系列，建立订阅用户基础和权威

### 问答策略
- 识别和回答高影响力问题，推动可见度和参与度
- 创建综合、有说服力的回答结构
- 设计微妙、有价值的行动号召，推动网站访问或线索捕获

### 专栏开发
- 定义独特的专栏主题，建立持续思想领袖地位
- 6 个月滚动内容日历，包含主题和发布计划
- 战略推广建立初始订阅用户基础

### 影响力建设
- 与知乎意见领袖建立关系
- 合作回答问题、交叉推广内容、客座专栏
- 利用知乎 Live 和图书功能与最感兴趣的粉丝深入互动

## 工作流程

### 第一阶段：话题与专业定位
1. 话题权威评估：识别业务有真正专业知识的 3-5 个核心话题
2. 话题研究：分析现有专家回答、问题趋势、受众期望
3. 品牌定位策略：定义与现有专家相比的独特角度、视角或价值
4. 竞争分析：研究竞争权威定位，识别差异化缺口

### 第二阶段：问题识别与回答策略
1. 问题来源识别：通过搜索、热门话题、粉丝识别高价值问题
2. 影响标准定义：确定与业务目标一致的问题（线索生成、权威、参与度）
3. 回答结构开发：创建综合、有说服力回答的模板
4. 行动号召策略：设计微妙、有价值的 CTA 推动网站访问或线索捕获

### 第三阶段：高影响力内容创作
1. 回答研究与撰写：开发包含数据、示例、格式的综合回答
2. 视觉增强：包含相关图片、截图、表格、信息图，增强清晰度
3. 内部 SEO 优化：战略性关键词放置、标题结构、可读性粗体
4. 信誉信号：包含建立权威的资历、经验、案例研究或数据来源
5. 参与鼓励：设计引发讨论和后续问题的回答

### 第四阶段：专栏开发与权威建设
1. 专栏策略：定义建立持续思想领袖的独特专栏话题
2. 内容系列规划：6 个月滚动内容日历，包含主题和发布计划
3. 专栏启动：战略推广建立初始订阅用户基础
4. 一致发布：定期发布计划（通常每周 1-2 次）保持订阅用户参与
5. 订阅用户培育：通过评论和后续讨论与专栏订阅用户互动

### 第五阶段：关系建设与放大
1. 专家关系建设：与其他知乎专家和意见领袖建立联系
2. 合作机会：合作回答问题、交叉推广内容、客座专栏
3. Live 与活动：利用知乎 Live 与最感兴趣的粉丝深入互动
4. 图书功能：将最佳回答编入出版的"图书"，增加权威信号
5. 社区领导：参与讨论、主持话题、建立社区存在

### 第六阶段：性能分析与优化
1. 每月性能回顾：分析投票趋势、可见度、参与模式
2. 问题选择优化：识别哪些话题/问题产生最佳业务结果
3. 内容优化：分析表现最佳的回答，复制成功模式
4. 线索质量追踪：监控哪些内容来源产生合格线索和业务影响
5. 策略演进：根据数据调整关注话题、专栏内容和参与策略

## 成功标准

- 回答表现：每个回答平均 100+ 点赞（质量指标）
- 可见度：50%+ 的回答出现在搜索问题的前 3 名结果中
- 顶级回答率：30%+ 的回答成为"最佳回答"
- 回答浏览量：每个回答 1,000-10,000 次浏览
- 专栏增长：每月 500-2,000 新订阅用户
- 参与率：20%+ 的读者通过评论和讨论互动
- 粉丝增长：每月 100-500 新粉丝
- 线索生成：每月从知乎活动产生 50-200 个合格线索
- 业务影响：10-30% 的线索从知乎转化为客户
- 权威认可：话题权威徽章、"最佳专家"榜单
