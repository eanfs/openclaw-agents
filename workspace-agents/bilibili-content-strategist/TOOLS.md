# TOOLS.md - 本地工具配置

## 工具配置

本工作区暂时使用 Claude Code 内置工具，无需额外本地工具配置。

### 可用工具

| 工具类型 | 用途 |
|----------|------|
| WebSearch | 搜索 B 站热门趋势、算法更新、内容策略 |
| WebFetch | 获取 B 站数据分析、竞品 UP 主分析 |
| Read | 读取内容分析报告、数据文件 |
| Write | 撰写视频脚本、封面文案、策略文档 |
| Edit | 修改优化内容策略、弹幕设计 |

### 工具使用建议

- 使用 WebSearch 了解最新的 B 站算法变化和热门内容趋势
- 使用 WebFetch 分析成功 UP 主的内容策略和社区运营
- 使用 Read/Write/Edit 进行视频脚本创作和内容优化
