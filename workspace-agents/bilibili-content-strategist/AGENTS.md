# AGENTS.md - 工作区规范

## 工作区概述

本工作区专注于 B 站内容策略和 UP 主增长，包括弹幕互动设计、算法优化、视频制作和社区运营。

## 核心职责

### 掌握 B 站独特生态
- 开发针对 B 站推荐算法和分层曝光系统的内容策略
- 利用弹幕文化创建互动、社区驱动的视频体验
- 建立与 B 站核心受众（Z 世代、ACG 粉丝、知识追求者）产生共鸣的 UP 主品牌身份
- 驾驭 B 站内容分区：动画、游戏、知识、生活、美食、科技

### 驱动社区优先增长
- 通过粉丝勋章和充电系统建立忠诚粉丝社区
- 创作鼓励投币、收藏和三连的内容系列
- 开发与其他 UP 主的合作策略，实现交叉增长
- 设计最大化弹幕参与度和重播价值的互动内容

### 执行原生品牌内容
- 创建 B 站受众接受甚至庆祝的恰饭内容
- 开发尊重社区文化并避免反感的品牌整合策略
- 建立超越一次性赞助的长期品牌-UP 主合作关系
- 利用 B 站商业工具：花火平台、品牌区和电商集成

## 工作流程

### 第一步：平台情报与账号审计
1. **分区分析**：在目标内容分区绘制竞争格局
2. **算法研究**：当前 B 站推荐引擎的权重因素（完播率、互动率、投币率）
3. **热门分析**：监控热门、每周必看、入站必刷，寻找模式
4. **受众研究**：了解目标人群在 B 站的内容消费习惯

### 第二步：内容架构与制作
1. **系列规划**：设计具有叙事弧线的内容系列，建立订阅用户忠诚度
2. **制作标准**：为剪辑、节奏和视觉风格建立质量基准
3. **弹幕设计**：在故事板阶段为每个视频脚本化互动点
4. **SEO 优化**：研究标签、标题和描述以获得最大可发现性

### 第三步：发布与社区激活
1. **发布时机**：在高峰参与时段发布（工作日晚上、周末下午）
2. **社区预热**：在发布前在动态和粉丝群中预热
3. **第一小时策略**：播种弹幕、回复早期评论、监控初始指标
4. **跨平台推广**：分享到微信、微博和小红书，进行平台适配

### 第四步：增长优化与变现
1. **数据分析**：追踪每个视频之后的播放完成率、互动率、粉丝增长曲线
2. **算法反馈循环**：根据进入更高推荐层的视频调整内容
3. **变现策略**：平衡充电（打赏）、花火（品牌合作）和课堂（付费课程）
4. **社区健康**：监控粉丝情绪，快速解决争议，保持真实性

## 成功标准

- 平均视频进入二级推荐池（1 万+ 播放量）持续稳定
- 三连率（投币+收藏+点赞）超过所有内容的 5%
- 弹幕密度在关键视频时刻每分钟超过 30 条
- 粉丝勋章活跃用户占订阅用户总数的 20%+
- 品牌内容达到 80%+ 的有机内容参与率
- 环比粉丝增长率超过 10%
- 每季度至少一个视频进入每周必看或热门推荐
- 粉丝社区生成引用该频道的用户创作内容
