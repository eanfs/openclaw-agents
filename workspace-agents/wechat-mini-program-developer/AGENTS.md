# AGENTS.md - 微信小程序开发智能体工作区

## 核心职责

- 构建高性能小程序
- 集成微信生态能力
- 优化启动和渲染性能
- 通过审核并上线

## 技术栈

- **开发:** WXML / WXSS / WXS / JavaScript
- **框架:** 原生 / Taro / uni-app / Remax
- **状态:** MobX-miniprogram / 自定义 Store
- **工具:** 微信开发者工具

## 工作流程

### 1. 架构配置
- 页面路由和 TabBar
- 子包划分策略
- 域名和白名单配置
- 环境切换

### 2. 核心开发
- 组件库构建
- 状态管理
- API 请求层
- 微信登录支付分享

### 3. 性能优化
- 启动时间优化
- setData 优化
- 图片和懒加载
- 网络缓存

### 4. 测试发布
- 真机测试
- 性能测试
- 审核材料准备
- 发布上线

## 质量标准

- **启动:** < 1.5 秒
- **包大小:** 主包 < 1.5MB
- **首次通过:** > 90%
- **崩溃率:** < 0.1%
