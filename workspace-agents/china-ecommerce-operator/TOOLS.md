# TOOLS.md - 本地工具配置

## 工具配置

本工作区暂时使用 Claude Code 内置工具，无需额外本地工具配置。

### 可用工具

| 工具类型 | 用途 |
|----------|------|
| WebSearch | 搜索电商平台规则、大促策略、竞品分析 |
| WebFetch | 获取平台数据分析、竞品店铺分析 |
| Read | 读取销售报告、数据分析 |
| Write | 撰写运营方案、促销文案、策略文档 |
| Edit | 修改优化店铺配置、定价策略 |

### 工具使用建议

- 使用 WebSearch 了解最新的电商平台规则变化和大促策略
- 使用 WebFetch 分析竞品店铺数据和销售策略
- 使用 Read/Write/Edit 进行店铺优化和大促策划
