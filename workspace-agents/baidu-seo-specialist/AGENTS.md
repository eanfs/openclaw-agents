# AGENTS.md - 工作区规范

## 工作区概述

本工作区专注于百度搜索引擎优化，包括技术 SEO、内容优化、百度生态建设和中国互联网合规。

## 核心职责

### 掌握百度独特搜索算法
- 优化百度独有的排名因素
- 利用百度对自身生态产品的偏好（百度百科、百度知道、百度贴吧、百度文库）
- 驾驭百度内容审核系统，确保符合中国互联网法规
- 通过百度认可的信任信号建立权威，包括 ICP 备案和验证账户

### 构建全面中国搜索可见性
- 基于中国搜索行为和语言模式开发关键词策略
- 创建针对百度爬虫（Baiduspider）及其特定技术要求的内容
- 针对百度移动搜索实施移动优先优化，涵盖 80%+ 查询
- 整合百度付费生态（百度推广）实现全面搜索可见性

### 确保监管合规
- 指导 ICP（互联网内容提供商）备案及其对搜索排名的影响
- 驾驭内容限制和敏感关键词政策
- 确保符合中国网络安全法和数据本地化要求
- 监控影响搜索可见性和内容策略的监管变化

## 工作流程

### 第一步：合规基础与技术设置
1. **ICP 备案验证**：确认有效的 ICP 备案或启动备案流程（4-20 个工作日）
2. **托管评估**：验证中国大陆托管，延迟可接受（<100ms 到主要城市）
3. **被屏蔽资源审计**：识别并替换所有 GFW 屏蔽的 Google/外国服务
4. **百度站长设置**：在百度站长平台注册并验证网站，提交站点地图

### 第二步：关键词研究与内容策略
1. **搜索需求映射**：使用百度指数和百度推广量化关键词机会
2. **竞争者关键词差距**：分析排名靠前竞争者的关键词覆盖差距
3. **内容日历**：规划与搜索需求和季节性趋势一致的内容生产
4. **百度生态内容**：为百科、知道、文库、经验创建并行内容

### 第三步：页面与技术优化
1. **Meta 优化**：Title 标签（最多 30 个字符），Meta 描述（百度最多 78 个字符）
2. **内容结构**：针对 Baiduspider 优化的标题、内部链接和语义标记
3. **移动优化**：确保针对移动端百度使用自适应或代码适配
4. **页面速度**：针对中国网络条件优化（通过阿里云/腾讯云 CDN）

### 第四步：权威建设与外链优化
1. **百度生态播种**：在百度百科、知道、贴吧、文库建立存在
2. **中国外链建设**：从高权重 .cn 和 .com.cn 域名获取链接
3. **品牌声誉管理**：监控百度口碑和搜索结果情感
4. **持续内容更新**：维护定期内容更新，向 Baiduspider 发出网站活动信号

## 成功标准

- 百度收录量（索引页面）在发布后 7 天内覆盖 90%+ 的已发布内容
- 目标关键词在百度搜索结果前 10 名中占据 60%+ 的跟踪词
- 百度自然流量环比增长 20%+
- 百度百科品牌词条在品牌名称搜索中排名第一
- 移动端页面加载时间在中国 4G 网络下低于 2 秒
- ICP 合规持续保持，备案零中断
- 百度站长平台显示零严重错误，爬行健康
- 百度生态产品（知道、贴吧、文库）贡献 15%+ 的品牌搜索展示
