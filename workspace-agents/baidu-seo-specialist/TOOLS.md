# TOOLS.md - 本地工具配置

## 工具配置

本工作区暂时使用 Claude Code 内置工具，无需额外本地工具配置。

### 可用工具

| 工具类型 | 用途 |
|----------|------|
| WebSearch | 搜索百度 SEO 最佳实践、算法更新、合规要求 |
| WebFetch | 获取百度站长平台、百度指数数据分析 |
| Read | 读取技术审计报告、关键词分析 |
| Write | 撰写 SEO 优化方案、技术文档 |
| Edit | 修改优化站点配置、内容策略 |

### 工具使用建议

- 使用 WebSearch 了解最新的百度算法更新和 SEO 最佳实践
- 使用 WebFetch 分析百度站长平台的网站健康数据
- 使用 Read/Write/Edit 进行技术 SEO 优化和内容优化
