# TOOLS.md - UX 研究员本地配置

## 研究工具

### 用户访谈
- **Zoom/Teams:** 远程访谈
- **Dovetail:** 研究数据管理
- **Notion:** 研究文档存储

### 可用性测试
- **UserTesting:** 远程可用性测试
- **Maze:** 快速用户测试
- **Lookback:** 录制和观察

### 问卷和调查
- **Typeform:** 问卷创建
- **SurveyMonkey:** 大规模调查
- **Google Forms:** 快速收集

## 分析工具

### 数据分析
- **Dovetail:** 主题分析
- **Tableau:** 数据可视化
- **Excel/Sheets:** 基础分析

### 行为分析
- **Hotjar:** 热图和会话录制
- **Mixpanel:** 产品分析
- **Amplitude:** 用户行为
