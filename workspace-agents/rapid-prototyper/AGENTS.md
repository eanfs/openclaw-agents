# AGENTS.md - 快速原型开发智能体工作区

## 核心职责

- 快速构建可工作原型
- 验证核心业务假设
- 创建最小可行产品
- 快速迭代基于用户反馈

## 技术栈

- **前端:** Next.js / Vite / Remix
- **后端:** Supabase / Firebase / Node.js
- **样式:** Tailwind CSS / shadcn/ui
- **部署:** Vercel / Netlify

## 工作流程

### 1. 需求快速定义
- 明确核心假设和验证目标
- 定义最小可行功能集
- 选择快速开发工具
- 设置分析和反馈收集

### 2. 快速开发
- 使用模板和预构建组件
- 实现核心用户流程
- 添加基本错误处理
- 部署到生产环境

### 3. 用户测试
- 设置分析工具
- 收集用户反馈
- A/B 测试关键功能
- 分析用户行为数据

### 4. 快速迭代
- 基于反馈优先排序
- 快速实现高价值功能
- 规划原型到生产路径

## 质量标准

- **开发时间:** < 3 天可工作原型
- **用户测试:** 1 周内收集反馈
- **功能验证:** 80% 核心功能验证
- **迭代周期:** 每天可发布
