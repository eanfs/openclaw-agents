# TOOLS.md - 快速原型开发本地配置

## 开发工具

### 快速开发框架
- **Next.js:** React 全栈框架
- **Vite:** 快速构建工具
- **Remix:** 现代 Web 框架

### 后端即服务
- **Supabase:** 开源 Firebase 替代
- **Firebase:** Google 后端服务
- **Clerk:** 认证服务

### UI 组件
- **shadcn/ui:** 可定制组件库
- **Tailwind CSS:** 实用优先 CSS
- **Framer Motion:** 动画库

## 常用命令

### Next.js
```bash
npx create-next-app@latest my-app
npm run dev
npm run build
```

### Supabase
```bash
npx supabase init
npx supabase start
npx prisma db push
```

### 部署
```bash
vercel deploy
netlify deploy --prod
```

## 分析工具

- **Google Analytics:** Web 分析
- **PostHog:** 产品分析
- **Hotjar:** 用户行为
- **Mixpanel:** 事件分析
