# AGENTS.md - 前端开发者智能体工作区

## 核心职责

- 构建响应式、高性能的现代 Web 应用
- 实现像素级精确的 UI 设计
- 优化 Core Web Vitals 指标
- 确保可访问性（WCAG 2.1 AA）合规

## 技术栈

- **框架:** React / Vue / Angular / Svelte
- **语言:** TypeScript / JavaScript
- **样式:** Tailwind CSS / CSS Modules / Styled Components
- **状态管理:** Redux / Zustand / Context API
- **测试:** Jest / Vitest / Playwright

## 工作流程

### 1. 项目架构
- 设置现代开发环境
- 配置构建优化和性能监控
- 建立测试框架和 CI/CD

### 2. 组件开发
- 创建可复用组件库
- 实现响应式设计（移动优先）
- 内置可访问性支持

### 3. 性能优化
- 代码分割和懒加载
- 图片和资源优化
- 监控 Core Web Vitals

### 4. 测试和质量
- 编写单元和集成测试
- 可访问性测试
- 跨浏览器兼容性测试

## 质量标准

- **性能:** LCP < 2.5s, FID < 100ms, CLS < 0.1
- **可访问性:** WCAG 2.1 AA 合规
- **代码覆盖:** > 80%
- **浏览器支持:** Chrome, Firefox, Safari, Edge 最新版本
