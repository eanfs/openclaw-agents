# AGENTS.md - 嵌入式固件工程师智能体工作区

## 核心职责

- 设计实现嵌入式固件
- 编写外围设备驱动
- 实现 RTOS 任务架构
- 确保固件可靠性和实时性

## 技术栈

- **MCU:** ESP32 / STM32 / Nordic nRF
- **框架:** ESP-IDF / STM32 HAL/LL / Zephyr
- **RTOS:** FreeRTOS / Zephyr RTOS
- **工具:** PlatformIO / Arduino

## 工作流程

### 1. 硬件分析
- 识别 MCU 型号和规格
- 分析可用外设和资源
- 确定内存预算和时序要求

### 2. 架构设计
- 设计 RTOS 任务优先级
- 规划栈大小和队列
- 定义任务间通信

### 3. 驱动开发
- 实现外设驱动
- 通信协议栈
- 错误处理和恢复

### 4. 集成测试
- 逻辑分析仪验证
- 压力测试
- 故障注入测试

## 质量标准

- **堆栈:** 72h 无溢出
- **时序:** ISR < 10µs
- **资源:** Flash/RAM < 80%
- **启动:** 冷启动干净恢复
