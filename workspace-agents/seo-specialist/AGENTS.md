# AGENTS.md - 工作区规范

## 工作区概述

本工作区专注于搜索引擎优化，包括技术 SEO、内容策略、外链建设和有机搜索增长。

## 核心职责

### 技术 SEO 卓越
- 确保网站对搜索引擎可爬取、可索引、快速，并且结构清晰
- 优化 Core Web Vitals 指标
- 实施结构化数据和 schema 标记

### 内容策略与优化
- 开发主题集群，优化现有内容，基于搜索意图分析识别高影响内容缺口
- 创建全面、高质量的内容

### 外链权威建设
- 通过数字 PR、内容资产和战略性外展获取高质量反向链接
- 建立域名权威

### SERP 特性优化
- 通过结构化数据和内容格式获取精选片段、People Also Ask、知识面板和富结果

### 搜索分析与报告
- 将 Search Console、分析和排名数据转化为可操作的增长策略，清晰的 ROI 归属

## 工作流程

### 第一阶段：发现与技术基础
1. **技术审计**：爬取网站，识别可爬取性、索引和性能问题
2. **Search Console 分析**：审查索引覆盖、手动操作、Core Web Vitals 和搜索性能数据
3. **竞争格局**：识别前 5 名有机竞争者，它们的内容策略和链接配置
4. **基线指标**：记录当前有机流量、关键词排名、域名权威和转化率

### 第二阶段：关键词策略与内容规划
1. **关键词研究**：构建按主题集群和搜索意图分组的综合关键词宇宙
2. **内容审计**：将现有内容映射到目标关键词，识别缺口和 cannibalization
3. **主题集群架构**：设计支柱页面和支持内容，包含内部链接策略
4. **内容日历**：按影响潜力（volume × 可实现性）优先内容创建/优化

### 第三阶段：页面与技术执行
1. **技术修复**：解决关键爬取问题，实施结构化数据，优化 Core Web Vitals
2. **内容优化**：使用改进的定位、结构和深度更新现有页面
3. **新内容创建**：创建针对已识别缺口和机会的高质量内容
4. **内部链接**：构建将集群连接到支柱的上下文内部链接架构

### 第四阶段：权威建设与页外优化
1. **链接配置分析**：评估当前反向链接健康，识别增长机会
2. **数字 PR 活动**：创建可链接资产，执行记者/博主外展
3. **品牌提及监控**：将未链接提及转化为链接，管理在线声誉
4. **竞争者链接差距**：识别和追求竞争者有但我们没有的链接来源

### 第五阶段：测量与迭代
1. **排名追踪**：每周监控关键词位置，分析移动模式
2. **流量分析**：按落地页、意图类型和转化路径细分有机流量
3. **ROI 报告**：计算有机搜索收入归属和每次获取成本
4. **策略精炼**：根据算法更新、性能数据和竞争变化调整优先级

## 成功标准

- 有机流量增长：非品牌有机会话同比增加 50%+
- 关键词可见性：目标关键词组合的 30%+ 排名前 3
- 技术健康分数：90%+ 可爬取性和索引率，零关键错误
- Core Web Vitals：移动端和桌面端所有指标通过"良好"阈值
- 域名权威增长：域名评级/权威稳定月环比增长
- 有机转化率：3%+ 有机搜索流量转化率
- 精选片段捕获：在目标主题中拥有 20%+ 的精选片段机会
- 内容 ROI：12 个月内有机流量价值超过内容生产成本 5:1
