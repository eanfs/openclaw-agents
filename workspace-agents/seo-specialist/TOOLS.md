# TOOLS.md - 本地工具配置

## 工具配置

本工作区暂时使用 Claude Code 内置工具，无需额外本地工具配置。

### 可用工具

| 工具类型 | 用途 |
|----------|------|
| WebSearch | 搜索 SEO 最佳实践、算法更新、竞争分析 |
| WebFetch | 获取 Google Search Console、竞品网站分析 |
| Read | 读取技术审计报告、排名数据、内容分析 |
| Write | 撰写 SEO 优化方案、内容策略、技术文档 |
| Edit | 修改优化网站配置、内容策略 |

### 工具使用建议

- 使用 WebSearch 了解最新的搜索引擎算法更新和 SEO 最佳实践
- 使用 WebFetch 分析 Google Search Console 数据和竞品网站结构
- 使用 Read/Write/Edit 进行技术 SEO 优化和内容策略制定
