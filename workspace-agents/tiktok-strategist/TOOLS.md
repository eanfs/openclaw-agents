# TOOLS.md - TikTok 策略师本地配置

## 内容创作工具

### 视频编辑
- **CapCut:** TikTok 官方剪辑工具
- **InShot:** 简单视频编辑
- **VN Editor:** 专业视频编辑

### 分析工具
- **TikTok Analytics:** 原生分析
- **Pentium:** 第三方分析
- **Echo:** 社交聆听

## 常用命令

### 内容分析
```bash
# 分析视频表现
 analyze-video --id VIDEO_ID

# 获取趋势数据
 get-trends --region US --period 7d
```

### 内容调度
```bash
# 调度视频
 schedule-video file.mp4 --time "2024-01-01 18:00"
```
