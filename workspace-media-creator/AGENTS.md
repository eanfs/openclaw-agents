# AGENTS.md

## 工作流程

```
用户输入 → 营销分析 → 脚本生成 → 素材创作 → 质量评估 → (循环优化) → 视频合成 → 多平台发布
```

## 核心 Agent

### 1. Market Agent (营销策划)
- 解析商品信息
- 市场分析、竞品研究
- 生成营销策略
- 输出：目标受众、卖点、情感基调、关键场景

### 2. Director Agent (视频导演)
- 生成视频脚本
- 创作视觉提示词
- 调用文生图/图生视频能力
- 输出：脚本、素材列表

### 3. Evaluate Agent (质量评估)
- 视觉质量评估 (75%权重)
- 内容相关度 (20%权重)
- 创意水平 (5%权重)
- 通过分数线：70分
- 最多优化5次

### 4. Release Agent (合成发布)
- FFmpeg 视频合成
- 添加字幕、音效、水印
- 多平台发布 (抖音、小红书、快手、B站、视频号)

## 质量标准

- 视觉质量：清晰度、色彩、构图、光影
- 内容相关度：关键词匹配、卖点体现
- 创意水平：新颖性、吸引力、差异化

## 场景类型

- Hook (开头)：3-5秒，痛点/好奇心
- Showcase (展示)：3-5秒/卖点，生活化语言
- CTA (行动召唤)：2-3秒，简洁有力

## 输出指标

- 视频时长：15-30秒
- 通过分数：≥70分
- 优化次数：最多5次
